# arc-index

Index and export utilities for research content.

## Features

- **rebuild** - Rebuild research indexes
- **stats** - Show index statistics

## Installation

```bash
go install github.com/mtreilly/arc-index@latest
```

## Usage

```bash
# Rebuild the index
arc-index rebuild

# Show statistics
arc-index stats
```

## Description

Arc-index generates index files (like `_INDEX.md`) from your research database, organizing papers, blog posts, and other items by type and date.
